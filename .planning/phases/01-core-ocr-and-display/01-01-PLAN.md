1. Run `npm init -y` in `/workspace` to create package.json.

2. Install dependencies:
   ```
   npm install tesseract.js
   npm install --save-dev esbuild eslint
   ```

3. Update package.json scripts section to:
   ```json
   "scripts": {
     "build": "node esbuild.config.js",
     "lint": "eslint extension/ --ext .js",
     "watch": "node esbuild.config.js --watch"
   }
   ```

4. Create `esbuild.config.js` that:
   - Bundles `extension/content/content.js` as an IIFE (content scripts cannot use ES modules) to `extension/dist/content.bundle.js`
   - Bundles `extension/popup/popup.js` as an IIFE to `extension/dist/popup.bundle.js`
   - Does NOT bundle the service worker (it is a simple relay, no imports needed)
   - Sets `bundle: true`, `minify: false` (dev), `sourcemap: true`, `target: ['chrome120']`
   - If `--watch` flag passed via process.argv, enable esbuild watch mode
   - The build output goes to `extension/dist/`

5. Create `.eslintrc.json`:
   ```json
   {
     "env": { "browser": true, "es2022": true, "webextensions": true },
     "parserOptions": { "ecmaVersion": 2022, "sourceType": "module" },
     "rules": {
       "no-unused-vars": "warn",
       "no-undef": "error",
       "semi": ["error", "always"]
     }
   }
   ```

6. Create the extension directory structure:
   ```
   mkdir -p extension/{popup,content,background,libs/tesseract,tessdata,icons,dist}
   ```

7. Copy tesseract.js runtime files from node_modules into the extension:
   - `node_modules/tesseract.js/dist/worker.min.js` -> `extension/libs/tesseract/worker.min.js`
   - `node_modules/tesseract.js/dist/tesseract.min.js` -> `extension/libs/tesseract/tesseract.min.js` (UMD build for content script)
   - The entire `node_modules/tesseract.js-core/` directory -> `extension/libs/tesseract-core/` (contains WASM files)

8. Download chi_sim traineddata (LSTM fast model, ~1.5 MB) for offline OCR:
   ```
   curl -L -o extension/tessdata/chi_sim.traineddata https://github.com/tesseract-ocr/tessdata_fast/raw/main/chi_sim.traineddata
   ```
   If the download fails, try the alternate URL: `https://raw.githubusercontent.com/tesseract-ocr/tessdata_fast/main/chi_sim.traineddata`

9. Create placeholder icon files (16x16, 48x48, 128x128 PNG). Use a simple script to generate minimal valid PNG files or create single-color placeholder PNGs. These are required by manifest.json.

10. Add a `build` npm script in package.json that also runs a copy step for the tesseract files (so `npm run build` is the single command to produce a loadable extension). Alternatively, add a `postinstall` script that copies tesseract files.

4. Create `extension/popup/popup.js`:
   - Implement `sendToContentScript(message)` using the pattern from RESEARCH.md Pattern 4:
     ```javascript
     async function sendToContentScript(message) {
       const [tab] = await chrome.tabs.query({ active: true, currentWindow: true });
       if (!tab?.id) return;
       try {
         return await chrome.tabs.sendMessage(tab.id, message);
       } catch (err) {
         console.warn('Content script not ready:', err);
       }
     }
     ```
   - btn-draw click handler: sends `{ action: 'ACTIVATE_DRAW_MODE' }` to content script. On success, update status dot to green (#22C55E), label to "Ready".
   - btn-recognize click handler: sends `{ action: 'RECOGNIZE' }`. On send, update status dot to amber (#F59E0B) with CSS pulsing animation, label to "Recognizing...". On response, revert to green/"Ready". Also sends `{ action: 'SHOW_PANEL' }` to ensure side panel is visible.
   - On popup load, send `{ action: 'GET_STATUS' }` to content script to check if a box is already drawn. If response says box exists, enable the recognize button and set status to "Ready"/green.
   - The recognize button starts as `disabled` attribute set. Remove disabled when box is drawn.

5. Create `extension/content/content.js` as a stub:
   ```javascript
   // Content script entry point - injected into YouTube pages
   // Modules will be imported here in subsequent plans

   let selectionBox = null; // Will hold selection coordinates {x, y, width, height}
   let sidePanelVisible = false;

   chrome.runtime.onMessage.addListener((message, sender, sendResponse) => {
     if (message.action === 'ACTIVATE_DRAW_MODE') {
       console.log('[YCR] Draw mode activated (stub)');
       sendResponse({ ok: true });
     }
     if (message.action === 'RECOGNIZE') {
       console.log('[YCR] Recognize triggered (stub)');
       sendResponse({ text: 'OCR not yet implemented' });
       return true; // async
     }
     if (message.action === 'GET_STATUS') {
       sendResponse({ boxDrawn: !!selectionBox });
     }
     if (message.action === 'SHOW_PANEL') {
       console.log('[YCR] Show panel (stub)');
       sendResponse({ ok: true });
     }
   });

   console.log('[YCR] YouTube Chinese Reader content script loaded');
   ```

   ```javascript
   chrome.runtime.onInstalled.addListener(() => {
     console.log('[YCR] YouTube Chinese Reader installed');
   });
   ```
