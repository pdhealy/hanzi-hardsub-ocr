Create `extension/content/sidepanel.js` as a module that exports a `SidePanel` class with the following API:

**Exported interface:**
```javascript
export class SidePanel {
  constructor()          // creates panel DOM, injects into page, initially hidden
  show()                 // makes panel visible
  hide()                 // hides panel (does not destroy)
  isVisible()            // returns boolean
  showLoading()          // shows "Recognizing..." spinner state
  showText(text)         // shows recognized OCR text
  showEmpty()            // shows "No text recognized" state
  showNoSelection()      // shows "No area selected" state
  showError(message)     // shows error state
  destroy()              // removes all DOM elements and listeners
}
```

**DOM structure:**
```
div#ycr-side-panel (fixed, right edge)
  div#ycr-resize-handle (left edge, 4px wide, full height)

    span#ycr-panel-title ("YouTube Chinese Reader")
    button#ycr-panel-close ("x", aria-label="Hide Panel")
  div#ycr-panel-content (scrollable output area)
    [dynamic content based on state]
```

**Left-edge resize handle (per D-12, RESEARCH.md code example):**
- `#ycr-resize-handle`: `position: absolute; left: 0; top: 0; width: 4px; height: 100%; cursor: col-resize; background: transparent; z-index: 1;`
- Add an invisible 8px hit area (4px handle + 4px padding using `::before` pseudo-element or by making the handle 8px wide but visually 4px)
- On hover: `background: #E5E7EB;` (subtle visual feedback)
- Resize interaction:
  ```javascript
  let isResizing = false;
  let startX, startWidth;
  handle.addEventListener('mousedown', (e) => {
    isResizing = true;
    startX = e.clientX;
    startWidth = parseInt(panel.style.width, 10) || 300;
    e.preventDefault();
    const onMouseMove = (e) => {
      if (!isResizing) return;
      const delta = startX - e.clientX;  // panel grows leftward
      const newWidth = Math.min(600, Math.max(200, startWidth + delta));
      panel.style.width = newWidth + 'px';
    };
    const onMouseUp = () => {
      isResizing = false;
      document.removeEventListener('mousemove', onMouseMove);
      document.removeEventListener('mouseup', onMouseUp);
    };
    document.addEventListener('mousemove', onMouseMove);
    document.addEventListener('mouseup', onMouseUp);
  });
  ```
  Min width: 200px, max width: 600px, default: 300px.

**All CSS must be injected via a `<style>` element** (not an external CSS file) because content scripts cannot easily link external stylesheets. Create a `<style id="ycr-panel-styles">` element and append it to `document.head`. This avoids conflicts with YouTube's styles.

**Namespace all CSS selectors** with `#ycr-` prefix to avoid collisions with YouTube's stylesheets. Do NOT use global element selectors.

**SPA navigation cleanup:**
- Listen for `yt-navigate-finish` and call `destroy()`.
- `destroy()` removes the panel element and the injected style element from the DOM.
